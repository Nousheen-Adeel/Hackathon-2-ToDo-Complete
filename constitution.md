# Project Constitution

This document defines the fundamental rules and guidelines for our project.

## Technology Stack
- Python version: 3.13
- Follow PEP 8 style guidelines for all code

## Storage Policy
- Use strictly in-memory storage (no databases yet)
- All data should be stored in variables/memory during execution
- No persistent storage mechanisms (files, databases, etc.) at this phase

## Development Principles
- Clean, readable code following Python best practices
- Proper documentation and comments where necessary
- Modular design with well-defined functions and classes
